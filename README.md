# Mobile-RE

<h1>IDA</h1>
<ol>
  <li>Install IDA onto your computer. Refer to the <a href="https://hcli.docs.hex-rays.com/getting-started/installation/">official guide</a> on how to do this.</li>
  <li>(Linux) The install path for IDA is located in: <code>Home\.local\share\applications\IDA</code></li>
  <li>Within the IDA folder, run the "ida" file for the GUI.</li>
</ol>
<br />

<h1>Mobile Security Framework (mobsf)</h1>
<ol>
  <li>To install mobsf, run the command: <code>sudo snap install mobsf</code></li>
  <li>mobsf can now be ran locally on your computer.</li>
</ol>
